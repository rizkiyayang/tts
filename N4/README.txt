TTS N4 by Belajar Wibu

Cara pakai:
1. Buka index.html atau tts-kosakata-jepang-n4.html lewat browser modern.
2. Jangan hapus atau pindahkan vocab.js dan favicon.png dari folder ini.
3. Klik Mulai Latihan untuk membuat puzzle baru.
4. Jika ada progress tersimpan, klik Lanjutkan Progress untuk meneruskan.

Input:
- Ketik romaji untuk mengisi hiragana.
- Contoh: mizu otomatis menjadi みず.
- Ketik nn untuk ん.
- Kana gabungan seperti cha, kyu, dan sho otomatis masuk satu kotak.
- Tekan Enter untuk cek jawaban.

Catatan progress:
- Puzzle yang belum selesai tersimpan di browser.
- Kosakata yang sudah selesai akan diprioritaskan agar tidak muncul lagi sampai stok hampir habis.
- Data progress tersimpan di localStorage browser pengguna.
